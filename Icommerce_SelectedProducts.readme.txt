Icommerce_SelectedProducts

Visar de N mest sålda/visade/tyngsta/längsta/... produkterna. Visar de med högst/
lägst värde på ett visst attribut.

- Special attribut: (ej vanliga produktattribut):
  - ordered_qty
  - views_count
  - weight
  - random

- I XML fil:
    <reference name="right">
        <block type="selectedproducts/collection" name="right.selected" template="icommerce/catalog/product/list/testselected.phtml" />
    </reference>

- Använd från .phtml fil: 
    $coll = $this->getCollection( "ordered_qty", 3 );
    foreach( $coll as $prod ){
        echo $prod->getName() . "<br>\n";
    }
